Automation
For Business

I've created a few different automation softwares for different use cases, in this article I'll elaborate, on the problem and solution for a few of them.

One of the automation software I have written using Python is a script used to fetch loan interest rates from a client credit union website. The script uses Beautiful Soup to extract the rates from a table on the website, and then updates the rate information in their text search engine ads. This increased the conversion rates by about 25%, as the ads were always displaying the most up-to-date and accurate information. The script also includes error handling and logging, ensuring that the process runs smoothly and any issues can be quickly identified and resolved.

Another automation software I have written using Python is a script that automates the process of creating tags for a client's website. The script uses Selenium to automatically fill out a web form with information from a CSV file, which saves hours of manual data entry. This automation not only cut down on time spent with compliance and submission, but also ensured that the tags were created accurately and consistently. The script also includes error handling and logging, to make sure the process runs smoothly and any issues can be quickly identified and resolved.

I have also written a script that automates the creation and management of Google Ads campaigns. The script uses the Google Ads API to create and update campaigns, ad groups, and ads, based on data from a CSV file. This automation saves a lot of time and effort, as the process of creating and managing campaigns can be quite time-consuming. This script also includes error handling and logging, to make sure the process runs smoothly and any issues can be quickly identified and resolved.
